About the OptiCrop Engine

A deep dive into our methodology, machine learning models, and classification metrics.

Project Overview

Modern farming faces massive challenges, including climate variability, soil degradation, and rising input costs. OptiCrop addresses these challenges by replacing guesswork with data-driven predictive models. The application processes soil nutrient levels and environmental climate markers to recommend the most optimal crop for sowing.

Supported Soil & Weather Parameters

Our machine learning engine requires seven key environmental and soil features to determine crop suitability:

Nitrogen (N) Ratio of Nitrogen content in soil (mg/kg)
Phosphorus (P) Ratio of Phosphorous content in soil (mg/kg)
Potassium (K) Ratio of Potassium content in soil (mg/kg)
Temperature Air temperature in degrees Celsius (°C)
Humidity Relative air humidity percentage (%)
pH Level Soil pH value indicating acidity/alkalinity
Rainfall Average seasonal rainfall in millimeters (mm)

Algorithm & Training Architecture

The core machine learning engine uses a multinomial Logistic Regression model trained on cross-verified agricultural datasets. Logistic Regression was selected for its high interpretability, rapid inference speed, and exceptional classification accuracy on clean soil boundary features.

Preprocessing pipelines include outlier trimming via the **Interquartile Range (IQR)** method applied to phosphorous metrics to ensure anomalous data points do not skew decision boundary margins. Features are split into 80% training and 20% test sets to validate model generalization capability.

Model Performance

97.8% Test Set Accuracy

The model was validated using stratified 5-fold cross-validation and achieved an overall F1-score of 98% across 22 crop classes.

Core Technologies

  • Scikit-Learn: ML model training, prediction API, and validation splits.
  • Pandas & NumPy: Statistical cleanup, outlier trimming, and array matrices.
  • Flask Backend: Robust Web WSGI server and API routers.
  • HTML5 / Vanilla CSS: Ultra-responsive layout with custom HSL styles.
  • Chart.js: Dynamic frontend research visualization.
